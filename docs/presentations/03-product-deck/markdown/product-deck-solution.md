## 📋 دستورالعمل طراحی برای Gamma.app

### تم و استایل
- **تم اصلی**: Glassmorphism (شیشه‌ای مات با افکت blur)
- **پس‌زمینه**: گرادیان آبی-سبز تیره (#0f172a → #1e3a5f → #134e4a)
- **کارت‌ها**: شیشه‌ای مات با blur، حاشیه سفید نیمه‌شفاف، سایه ملایم
- **رنگ اصلی**: فیروزه‌ای (#14b8a6) و آبی (#0ea5e9)
- **رنگ تاکیدی**: زرد (#eab308) برای هایلایت، سبز (#22c55e) برای موفقیت
- **فونت فارسی**: وزیرمتن یا یکان (ضخامت‌های مختلف)
- **جهت**: راست‌به‌چپ (RTL)
- **آیکون‌ها**: خطی (outline) با رنگ سفید یا فیروزه‌ای

### قوانین کلی
- این ارائه محصول‌محور است و روی امکانات و صفحات تمرکز دارد
- استفاده از Wireframe‌های ساده برای نمایش صفحات
- نمایش مسیرهای کاربری (User Flows)
- انیمیشن‌های ظریف fade-in برای ورود عناصر

```
✅ داخل محدوده (In Scope) — MVP

• ثبت کیلومتر فعلی و به‌روزرسانی آن
• پشتیبانی از چند خودرو

زیرساخت:
• حساب کاربری + ذخیره‌سازی ابری
• کانال‌های یادآوری: داخل اپ + ایمیل
• رابط کاربری فارسی (راست‌چین + تقویم شمسی)
• وب‌اپلیکیشن واکنش‌گرا (موبایل، تبلت، دسکتاپ)
```

```
❌ خارج از محدوده (Out of Scope) — فازهای بعدی

فاز ۲ (۶-۱۲ ماه):
⏳ تبدیل به PWA + قابلیت نصب
⏳ کار آفلاین (Offline-first)
⏳ Push Notifications برای مرورگرها
⏳ گزارش‌گیری پیشرفته و نمودارها
⏳ تمپلیت‌های آماده سرویس‌های بیشتر
⏳ پشتیبانی از چند کاربر (خانواده)
⏳ یادآوری بر اساس مکان

🔮 اتصال به تعویض روغنی‌ها و اتوسرویس‌ها
🔮 رزرو نوبت آنلاین
```
