# خودروبان (KhodroBan) - خلاصه سند

**شعار:** کاهش هزینه‌های نگهداری خودرو با هوش مصنوعی

**هدف:** مدیریت خودرو، ثبت سرویس‌های دوره‌ای، پیگیری هزینه‌ها و دریافت مشاوره هوشمند برای کاهش هزینه‌ها

**صفحات MVP:**

1. Landing Page
2. Dashboard (وضعیت خودروها و هشدارها)
3. مدیریت خودروها
4. ثبت سرویس/هزینه
5. گزارش‌ها (CSV/PDF)
6. مشاور هوشمند AI (چت‌بات با Gemini/OpenAI)
7. تنظیمات
8. نوتیفیکیشن‌ها

**معماری:**

- Frontend: SvelteKit
- Backend: Supabase (PostgreSQL + Realtime)
- یادآوری: Python Cron Job (روزانه ساعت ۸)
- AI: Gemini/OpenAI برای مشاوره شخصی‌سازی شده

**طراحی:** Glassmorphism، Mobile-First، RTL فارسی، فونت Vazirmatn

**ویژگی‌های کلیدی:**

- یادآوری هوشمند ۷ روز قبل از موعد سرویس
- Real-time updates
- مشاور AI با دسترسی به سوابق کاربر
- مدیریت چند خودرو
- گزارش‌گیری هزینه‌ها
