هدف
ایجاد اسکریپت sql که از روی مستند پیوست‌شده (PDF/DOCX/Markdown/متن ساده) یک پایگاه‌داده کامل تولید کند: شامل استخراج ساختار (جداول، ستون‌ها، انواع داده، کلیدها، ایندکس‌ها)، تولید اسکیمای SQL، و در صورت وجود داده‌های نمونه، تولید دستورات INSERT یا بارگذاری داده. خروجی باید قابل اجرای مستقیم برای ساخت پایگاه‌داده (فایل .sql یا ایجاد یک فایل پایگاه‌داده مثل SQLite) و شامل گزارش موارد نامشخص برای بازبینی دستی باشد.
الزامات سخت

- خروجی قطعی: یک فایل SQL استاندارد + در صورت انتخاب sqlite، فایل پایگاه‌داده نهایی.
- idempotent: اجرای مجدد نباید خطا دهد (استفاده از IF NOT EXISTS / transactions).
- گزارش خطا و موارد نامشخص: تولید فایل log و یک فایل review.txt برای المان‌هایی که به‌صورت خودکار تشخیص داده نشدند.
- نگاشت انواع داده از متن به SQL: حداقل انواع int, bigint, float, decimal, varchar/text, boolean, date, datetime. برای موارد مبهم، مقدار پیش‌فرض قابل تنظیم یا null قرار دهد.
- پشتیبانی از کلید اصلی، کلید خارجی، یونیک و ایندکس‌ها در صورت توضیح در مستند.
  راهنمایی
- در مواجهه با ابهام‌های ساختاری یا اصطلاحات غیررسمی، از یک پیش‌فرض محافظه‌کار (nullable و varchar(255)) استفاده کند.
