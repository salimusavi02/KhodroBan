# صفحه ارتقا به Pro

این صفحه برای ارتقای کاربران از نسخه رایگان به نسخه‌های Pro و Pro+ طراحی شده است.

## ساختار

```
/upgrade/
├── +page.svelte              # صفحه اصلی
└── README.md                 # این فایل
```

## کامپوننت‌ها

### UpgradeHero
هدر صفحه با انیمیشن ستاره و توضیحات جذاب.

### PricingCard
کارت‌های قیمت‌گذاری برای انتخاب طرح Pro یا Pro+.

### FeatureComparison
جدول مقایسه ویژگی‌های Free vs Pro vs Pro+.

### FAQSection
سوالات متداول درباره ارتقا.

## سرویس‌ها

### upgradeService.ts
مدیریت ارتقا و ارتباط با backend:
- `upgrade(plan)`: ارتقا به طرح جدید
- `getCurrentPlan()`: دریافت پلان فعلی
- `checkPaymentStatus()`: بررسی وضعیت پرداخت
- `cancelSubscription()`: لغو اشتراک

## مسیرها

- `/upgrade` - صفحه ارتقا

## ویژگی‌ها

- ✅ طراحی واکنش‌گرا (Mobile-First)
- ✅ انیمیشن‌های ظریف
- ✅ دسترسی‌پذیری کامل (WCAG 2.1 AA)
- ✅ Glassmorphism design
- ✅ انتخاب طرح Pro یا Pro+
- ✅ مقایسه ویژگی‌ها
- ✅ سوالات متداول
- ✅ اتصال به درگاه پرداخت (Mock)

## نحوه استفاده

1. کاربر به `/upgrade` می‌رود
2. طرح مورد نظر را انتخاب می‌کند
3. روی دکمه "شروع ارتقا" کلیک می‌کند
4. به درگاه پرداخت منتقل می‌شود (در نسخه واقعی)
5. پس از پرداخت موفق، پلان به‌روزرسانی می‌شود

## توسعه

برای اضافه کردن ویژگی‌های جدید:
1. کامپوننت‌های موجود را ویرایش کنید
2. سرویس upgradeService را به‌روزرسانی کنید
3. صفحه +page.svelte را متناسب با نیاز تغییر دهید

## تست

```bash
cd frontend
npm run dev
# به http://localhost:3000/upgrade بروید
```
